La difference entre put et patch
put : Lorsqu'on veut modifier entièrement une entrée dans la BD.
patch : Lorsqu'on veut modifier une partie de l'entré.

C => Create, Créer -> Requete de type POST
R => Read, Lire -> GET
U => Update, Modifier -> PUT|PATCH
D => Delete, Supprimer -> DELETE
